Question-1: What are the different ways to select an element in the DOM?

getElementById("id") নির্দিষ্ট id অনুযায়ী এলিমেন্ট সিলেক্ট করতে।
getElementsByClassName("class") নির্দিষ্ট class অনুযায়ী একাধিক এলিমেন্ট সিলেক্ট করতে।
getElementsByTagName("tag") নির্দিষ্ট HTML ট্যাগ অনুযায়ী একাধিক এলিমেন্ট সিলেক্ট করতে।
querySelector("selector") প্রথম করা CSS সিলেক্টর অনুযায়ী এলিমেন্ট সিলেক্ট করতে।
querySelectorAll("selector") নির্দিষ্ট CSS সিলেক্টর অনুযায়ী সব মিল থাকা এলিমেন্ট সিলেক্ট করতে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML > HTML ট্যাগসহ নেয় innerText > শুধু দৃশ্যমান টেক্সট নেয় textContent লুকানো টেক্সটসহ সব নেয় কিন্তু HTML ট্যাগ উপেক্ষা করে।

Question-3: What is event delegation in the DOM?

Event Delegation হলো একটি মূল (parent) এলিমেন্টে ইভেন্ট যুক্ত করা যাতে তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট হ্যান্ডল করা যায়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো যখন একটি চাইল্ড এলিমেন্টে ইভেন্ট ট্রিগার হলে সেটি তার প্যারেন্ট ও উপরের এলিমেন্টগুলোতে ধাপে ধাপে পৌঁছে যায়।

Question-5: How do you create, add, and remove elements using JavaScript?

createElement(), appendChild(), insertBefore(), removeChild(), এবং remove() ব্যবহার করে এলিমেন্ট তৈরি, যোগ ও মুছা যায়।
