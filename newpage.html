Question-1: What are the different ways to select an element in the DOM?

The Document Object Model(DOM)is an important concept in web development, representing the structure of a web page. Using JavaScript, it is possible to select different elements in the DOM and interact with these selected elements. There are several methods for selecting an element in the DOM, which can be useful in different situations.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

DOM manipulation is important in web development, and innerHTML, innerText, and textContent are used to extract text or HTML content from the DOM. Although the work may seem very similar but the functionality may be different, there are some important differences.

Question-3: What is event delegation in the DOM?

Events are an important concept in web development, which are used to perform specific actions based on various user actions (such as clicking, typing, scrolling, etc.). But sometimes when you have to add event listeners on many elements, it becomes complicated and can affect performance. Event Delegation is used to solve this problem.

Question-4: What is event bubbling in the DOM?

Events are an important topic in web development, which are created to describe various user actions such as clicking, typing, scrolling lines. When an event occurs, it can appear in different layer(s) of the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

In web development, the DOM (Document Object Model) is the structure of a webpage that can be modified using JavaScript. You can create new elements, add and remove existing elements using JavaScript.
